ОТЗЫВ РУКОВОДИТЕЛЯ

на дипломный проект по теме "Веб-платформа для подбора туристических маршрутов с персональными рекомендациями"

Представленный дипломный проект посвящён разработке веб-платформы Tourist для планирования туристических маршрутов с учётом индивидуальных предпочтений пользователя. Тема работы является актуальной, так как пользователю требуется удобный инструмент для выбора достопримечательностей, построения маршрута и сохранения готового плана поездки.

В ходе выполнения проекта студентом была разработана клиент-серверная система, включающая регистрацию и авторизацию пользователей, настройку профиля, отображение туристических объектов на интерактивной карте, формирование персональных рекомендаций, построение маршрута, расчёт расстояния и длительности, а также сохранение и удаление маршрутов. Дополнительно реализован административный раздел для просмотра пользователей и их маршрутов.

Клиентская часть проекта выполнена с использованием React и TypeScript, серверная часть реализована на Node.js с применением Express. Для хранения данных используется PostgreSQL, для отображения карты применяются Leaflet и OpenStreetMap. Выбранные технологии соответствуют поставленным задачам, а структура приложения разделена на логические модули, что упрощает сопровождение и дальнейшее развитие системы.

При выполнении дипломного проекта студент продемонстрировал умение анализировать предметную область, проектировать структуру веб-приложения, работать с базой данных, реализовывать серверное API и создавать пользовательский интерфейс. Разработанная система позволяет пройти полный пользовательский сценарий: от входа в профиль и настройки предпочтений до построения и сохранения маршрута.

Дипломный проект соответствует заявленной теме, поставленная цель достигнута, основные задачи выполнены. Считаю, что дипломник заслуживает присвоения квалификации «инженер-системотехник».
